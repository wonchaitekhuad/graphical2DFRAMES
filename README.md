## Requirements
- Visual Studio 2019 or later
- .NET Framework 4.7.2

## Project Structure
- `Form1.cs` - Main application form
- `Forms2/` - Additional dialog forms
- `Class1.cs` - Core classes and data structures
- `LU.cs` - LU decomposition implementation
- `Resources/` - Application resources (images, icons)
- `Properties/` - Project properties and settings

## Building
Open `Graphical_2D_Frame_Analysis_CSharp.sln` in Visual Studio and build the solution.
